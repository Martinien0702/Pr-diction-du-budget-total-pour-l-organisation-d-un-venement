# Pr-diction-du-budget-total-pour-l-organisation-d-un-venement
Cette application permet d’obtenir rapidement une estimation du budget total d’un événement à partir d’un ensemble de caractéristiques. Elle s’adresse aux organisateurs d’événements, gestionnaires de projets, agences ou analystes souhaitant obtenir une estimation fiable avant de préparer un devis complet.
